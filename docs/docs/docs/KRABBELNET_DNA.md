
# 🧡 KrabbelNet DNA

> Het internet mag weer gezellig zijn.

Dit document bevat de ontwerpprincipes en UX-ideeën die KrabbelNet onderscheiden. Niet alles wordt direct gebouwd, maar alles in dit document moet bijdragen aan het gevoel van KrabbelNet.

---

# Ontwerpprincipes

- Mensen boven functies.
- Gezelligheid boven engagement.
- Rust boven drukte.
- Privacy is vanzelfsprekend.
- Begroet mensen als vrienden, niet als gebruikers.
- Vier kleine momenten.
- Laat gebruikers glimlachen.

# UX-regel

Elke nieuwe functie moet bijdragen aan het gevoel:

"Het internet mag weer gezellig zijn."
💛 Animatieprincipes
Animaties duren maximaal 1 seconde.
Animaties mogen warmte toevoegen, nooit afleiden.
Het logo mag één rustige hartslag geven bij binnenkomst.
Nieuwe content mag zacht verschijnen (fade/slide).
Geen schreeuwerige of verslavende effecten.
