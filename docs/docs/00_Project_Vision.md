# 🧡 Project Vision

## Project

KrabbelNet

---

## Missie

KrabbelNet brengt het sociale internet terug naar de mensen.

Geen algoritmes.

Geen eindeloze reclame.

Geen manipulatie.

Gewoon vrienden.

---

## Visie

Wij geloven dat sociale media weer mogen voelen als vroeger.

Niet omdat vroeger beter was.

Maar omdat mensen zich toen meer verbonden voelden.

KrabbelNet wordt een plek waar mensen zichzelf kunnen zijn, herinneringen kunnen delen en elkaar op een persoonlijke manier kunnen ontmoeten.

---

## Onze slogan

> Het internet mag weer gezellig zijn.
